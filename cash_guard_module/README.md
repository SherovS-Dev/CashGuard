# CashGuard - Приложение для управления финансами

Безопасное приложение для управления личными финансами с биометрической аутентификацией.

## Структура проекта

```
cash_guard/
├── lib/
│   ├── main.dart                          # Точка входа приложения
│   ├── models/
│   │   └── user_model.dart                # Модели данных пользователя и банковских карт
│   ├── services/
│   │   ├── secure_storage_service.dart    # Сервис для безопасного хранения данных
│   │   └── biometric_auth_service.dart    # Сервис биометрической аутентификации
│   └── screens/
│       ├── lock_screen.dart               # Экран блокировки с аутентификацией
│       ├── user_setup_screen.dart         # Экран первоначальной настройки
│       └── home_screen.dart               # Главный экран с финансами
└── pubspec.yaml                           # Зависимости проекта
```

## Функционал

### 1. Аутентификация (lock_screen.dart)
- Создание и ввод пароля
- Биометрическая аутентификация (отпечаток пальца / Face ID)
- Автоматический запуск биометрии при старте
- Безопасное хранение пароля (SHA-256 хеш)

### 2. Настройка профиля (user_setup_screen.dart)
- Ввод имени пользователя
- Указание суммы наличных
- Добавление нескольких банковских карт:
    - Название карты
    - Название банка (необязательно)
    - Последние 4 цифры карты
    - Баланс на карте

### 3. Главный экран (home_screen.dart)
- Отображение общего баланса
- Информация о наличных
- Список банковских карт с балансами
- Возможность редактирования профиля
- Сброс пароля и данных

## Модели данных

### User (user_model.dart)
- `name` - имя пользователя
- `cashInHand` - наличные деньги
- `bankCards` - список банковских карт
- `totalBalance` - автоматический расчет общего баланса

### BankCard (user_model.dart)
- `cardName` - название карты
- `cardNumber` - последние 4 цифры
- `balance` - баланс на карте
- `bankName` - название банка (необязательно)
- `maskedCardNumber` - маскированный номер (**** **** **** 1234)

## Сервисы

### SecureStorageService
- Хеширование и проверка паролей
- Сохранение и загрузка данных пользователя
- Работа с flutter_secure_storage

### BiometricAuthService
- Проверка доступности биометрии
- Получение списка доступных типов биометрии
- Аутентификация через биометрию
- Обработка ошибок биометрии

## Установка

1. Убедитесь, что Flutter установлен
2. Установите зависимости:
```bash
flutter pub get
```

3. Для Android добавьте в `android/app/src/main/AndroidManifest.xml`:
```xml
<uses-permission android:name="android.permission.USE_BIOMETRIC"/>
<uses-permission android:name="android.permission.USE_FINGERPRINT"/>
```

4. Для iOS добавьте в `ios/Runner/Info.plist`:
```xml
<key>NSFaceIDUsageDescription</key>
<string>Используется для входа в приложение</string>
```

5. Запустите приложение:
```bash
flutter run
```

## Безопасность

- Пароли хешируются с использованием SHA-256
- Данные хранятся в flutter_secure_storage (AES-256 шифрование)
- Биометрическая аутентификация использует нативные API устройства
- Все финансовые данные хранятся локально и зашифрованы
